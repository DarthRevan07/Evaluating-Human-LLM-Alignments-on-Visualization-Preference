# Supplementary PDFs

This directory contains PDF documents that provide visual and descriptive overviews of key aspects of the study **"Evaluating Human-LLM Alignments on Visualization Preference"**.

## Files

| File | Description | Pages |
|------|-------------|-------|
| `annotation_ui_overview.pdf` | Overview of the web-based annotation UI used to collect human preference judgments | 3 |
| `annotator_grading_samples.pdf` | Curated samples of annotator preference judgments, including human-LLM comparison tables | 4 |

## annotation_ui_overview.pdf

Covers:
- The annotation platform's technology stack and hosting setup
- Step-by-step study flow (consent, training, annotation tasks, post-task survey)
- Per-pair interface controls (preference radio buttons, confidence slider, free-text rationale, flag button)
- Quality control measures (attention checks, time filters, intra-annotator consistency pairs)
- Annotator demographics and compensation details

## annotator_grading_samples.pdf

Contains grading samples for five representative visualization pairs, each including:
- A description of both visualizations in the pair (design dimension, chart type, encoding)
- Distribution of human preference votes with inter-annotator agreement scores (Krippendorff's alpha)
- Representative annotator rationale quotes (verbatim, anonymized)
- LLM preference judgments per model, prompt condition, and confidence level
- A summary alignment table comparing human and LLM majority preferences across all sample pairs
