# Evaluating-Human-LLM-Alignments-on-Visualization-Preference

This repository houses the supplementary material for the submission **"Evaluating Human–LLM Alignments on Visualization Preference"** at **IEEE VIS 2026 (Short Papers Track)**.

## Acknowledgements & Data Source

Parts of the dataset used in this work are derived from the dataset released with the paper **“C2: Scalable Auto-Feedback for Charts”** and its accompanying repository. We gratefully acknowledge the authors for making their data and resources publicly available.

If you use this repository, please also cite the original *C2* paper and dataset.
